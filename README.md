# Cranium

An awesome ETL tool.

## Installation

Add this line to your application's Gemfile:

    gem 'cranium'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cranium

## Development

start up the db:

    $ docker-compose up -d
